# Log Format

## Target File

- `E:\RDR2 RPG MODS\개발\TASK_COMPLETION_LOG.md`

## Required Fields

- timestamp
- title
- request
- result
- skills used
- files touched
- validation
- residual risk

## Style

- Keep entries concise but concrete.
- Use project-relative paths in the log body when helpful.
- Say `analysis only` if no files changed.
- Say `not run` for validations that were intentionally skipped.
